代码库

想要快速的使用和风天气的数据吗？我们提供了一些官方示例代码和非官方代码库，可以让你的开发工作更加简单快速。

对于不同的项目、产品和开发人员，调用方式可能会有所不同，你可能还需要在此基础上做出一些修改和优化，以便更加适合你的开发项目。

如果有你不错的代码希望分享，可以发送邮件 support@heweather.com 给我们（请注明作者名称、项目名称、简短说明、仓库地址、主要语言）🙏❤️👍

我们不对非官方代码进行任何技术支持，也无法确保非官方代码的有效性

C

  • heweather-api-c @笑乐 如何使用 C语言访问和风天气的数据接口，http + ssl

C#

  • MyWeather UWP @Grergo UWP天气应用

C++

  • Weather @JX Cheng Weather 是使用 C++ & Qt Quick 开发的一款天气 App，理论上可以在 Windows、Mac OS、Linux、Android、iOS 等平台上运行。
  • ESP8266_Heweather @Ldufan Arduino开发平台使用ESP8266获取和风天气的第三方库

Dart

  • OpenWeatherPlus-Flutter @Daniel Dai Opensource weather app built with Flutter & fish-redux, backed by free weather data from heweather.com (用Flutter和fish-redux开发的开源天气APP)

  • go-heweather @Ink-33和风天气的非官方Golang SDK

JavaScript

  • Météo 天气 @Venthan a MBE style weather web based on VUE
  • 基于和风天气api和vue.js @ruiruiruiruirain 基于和风天气api和vue.js

Python

  • HeWeather @Acring 和风天气python接口实现
