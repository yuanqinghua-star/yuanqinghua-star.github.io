雷达图

卫星云图和雷达图，包括中国风云4号卫星云图、美国GEOS卫星云图、日本向日葵8号卫星云图，所获取的图片均为当前请求时间最新的图片。如需要批量下载或指定时间，请与我们联系。

请求URL

# 商业版
https://api.heweather.net/v7/radar?{parameters}

请求参数

请求参数包括必选和可选参数，如不填写可选参数将使用其默认值，参数之间使用&进行分隔。
